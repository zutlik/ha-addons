# Claude Code - Home Assistant Add-on

Run [Claude Code](https://claude.ai/code) as a persistent AI agent on your Home Assistant device (Raspberry Pi or other).

## Features

- **Always on** — runs continuously, restarts automatically with Home Assistant
- **Telegram** — send prompts and receive responses via your own Telegram bot
- **Remote control** — access your session from anywhere via claude.ai/code
- **Web terminal** — browser-based terminal for login and interactive use
- **Smart updates** — automatically installs new Claude Code versions without rebuilding the image
- **Memory** — maintains context across restarts via `memory.md`

## Requirements

- Home Assistant OS or Supervised
- Claude.ai **Pro or Max** subscription
- Raspberry Pi 4/5 (aarch64) or any amd64 machine

## Installation

Add this repository to your Home Assistant add-on store, then install **Claude Code**.

See [DOCS.md](DOCS.md) for setup instructions.
